◆ generateESK()

int generateESK ( void  )

Checkes the ESK key files and perform following tasks if the files are not exist.

  • generates a random HMAC AES key
  • generate a random RSA 3072 key pair
  • create the DS encrypted data from RSA and HMAC key
  • blow the HMAC efuse
  • generate a CSR for the RSA key
  • zero all the storage used by HMAC and RSA private key
  • saves the CSR, RSA public key, and DS encrypted data in NVS
    Note
    Burns the EFUSE when the new keys are being generated.
    Returns
    SUCESS/FAILURE on key generation
