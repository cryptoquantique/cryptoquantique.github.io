Detailed Description

Contains all the functions related to the crypto environment/technology used.

When adding support for a new crypto technology, it is particularly important to implement the functions below:

  • cryptoInit()
  • getESK()
  • getDSK()
  • newDSK()
  • createKey()
  • freeKey()
